1. The code folder contains the code for all the models.
2. Q1_cbow.py, Q1_lstm.py, Q1_skipgram.py contains code for Bag-of-words, lstm and skipgram word2vec models respectively.
3. Q2_sentiment_cbow.py, Q2_sentiment_lstm.py, Q2_sentiment_skipgram.py contains sentiment analysis code for Bag-of-words, lstm and skipgram word2vec models respectively.
4. Q2_sentiment_pretrained_glove.py contains sentiment analysis for pretrained glove embeddings.
5. Q2_sentiment_pretrained_gensim.py contains sentiment analysis for embeddings trained on gensim word2vec model.
